# NAME

**verifier** - Starts the Verifier gRPC client

# SYNOPSIS

**verifier** [*OPTIONS*] [*ARGUMENT*]

# DESCRIPTION

**verifier** - Performs Remote Attestation with the specified Attestor.


# OPTIONS

  * **-a**, **\--attestor**:

    The IP address, host name, or DNS name of the attestor gRPC service
